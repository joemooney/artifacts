# artifacts

Shareable **rendered deliverables** for the AIDA project — PDFs, slide exports, and other
binaries meant to be linked and shared, kept *out* of the code repo so it stays source-only.

**Sources live in `joemooney/aida`** (markdown under `docs/`). The renders here are derived
(e.g. `pandoc <doc>.md -o <doc>.pdf`) — regenerate from source rather than hand-editing.

## Contents
- `competitive-analysis/` — rendered competitive-analysis PDFs (source: aida `docs/competitive-analysis/`)

## Sharing
Public repo → files are linkable directly. (Optional level-up: front with a Cloudflare
custom domain, e.g. `share.joemooney.com`, for branded permanent URLs.)
